Tic Tac Toe Game in C
A clean and beginner-friendly console-based Tic Tac Toe game developed using C language.
The project focuses on core programming concepts like arrays, functions, and game logic.



. Features
Two-player mode (Player X vs Player O)
3×3 interactive game board
Automatic winner and tie detection
Simple terminal UI
No external libraries required
Compatible with Visual Studio, GCC, and other C compilers



.  Technologies Used
Language: C
IDE: Visual Studio / GCC
Concepts: Arrays, Loops, Functions, Conditional Logic

. How to Run



Using Visual Studio
Create a Console App (C/C++)
Paste the code into TicTacToe.c
Set Compile as C (/TC)
Press Ctrl + F5
Using GCC
Copy code
Bash

./TicTacToe
.  Gameplay Instructions
Player X starts first
Enter row and column numbers (1–3)
First player to align three symbols wins
If all cells are filled → Draw
